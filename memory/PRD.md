# Презентация КП для ПАО «Ростелеком» — Hop.Agency

## Описание
Веб-приложение для просмотра и скачивания коммерческого предложения Hop.Agency для ПАО «Ростелеком» в формате слайдовой презентации.

## Функционал
- 14 слайдов с контентом из КП
- Навигация: стрелки клавиатуры, кнопки, точки прогресса, свайпы
- **Скачивание PDF через @react-pdf/renderer** — прямая генерация векторного PDF без html2canvas
- Полноэкранный режим
- Плавные анимации переходов (framer-motion)

## Технический стек
- React + Tailwind CSS + shadcn/ui
- framer-motion для анимаций веб-презентации
- **@react-pdf/renderer** для генерации PDF (заменил html2canvas + jsPDF)
- Шрифт Inter (с кириллицей) для PDF

## Структура проекта
```
/app/frontend/src/
├── components/
│   ├── PdfGenerator.jsx       ← PDF-генерация (@react-pdf/renderer)
│   ├── slides/                ← Веб-слайды (14 компонентов)
│   └── ui/                    ← shadcn компоненты
└── pages/
    └── Presentation.jsx       ← Основной компонент
```
