# Презентация КП для ПАО «Ростелеком» — Hop.Agency

## Описание
Веб-приложение для просмотра и скачивания коммерческого предложения Hop.Agency для ПАО «Ростелеком» в формате слайдовой презентации. Визуальный стиль вдохновлён сайтом hop.agency (глубокий тёмно-синий фон, коралловый акцент, типографика Space Grotesk + Inter).

## Функционал
- 14 слайдов с контентом из КП
- Навигация: стрелки клавиатуры, кнопки, точки прогресса, свайпы
- Скачивание PDF через кнопку (window.print)
- Полноэкранный режим
- Плавные анимации переходов (framer-motion)
- Адаптивная вёрстка

## Технический стек
- React + Tailwind CSS + shadcn/ui
- framer-motion для анимаций
- Design tokens в HSL формате

## Структура слайдов
1. Титульный слайд
2. Контекст и основание
3. Предмет предложения (3 фокуса)
4. Цель программы
5. Задачи программы (7 задач)
6. Пакет готовности к рынку (9 компонентов)
7. Сводные результаты (3 направления)
8. Почему Hop.Agency (статистика)
9. Содержание работ — этапы 1–4
10. Содержание работ — этапы 5–8
11. Команда проекта (5 специалистов)
12. Дорожная карта (12 недель)
13. Стоимость (2 975 000 ₽, скидка 15%)
14. Контакты
